# symfony-api

Set up symfony with docker

## Setup steps history

- Install `php` through `homebrew`
- Install `xdebug` through `pecl`
- Install `symfony` through `homebrew`
- Run `symfony new src`
- Create `docker` folder
- Create `Dockerfile`
- Create `nginx` folder with `default.conf`
- Create `entrypoint.sh`
- Create `.env.example`
- Create `Makefile`
